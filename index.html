Docker

Is the answer to everything!


						docker run containers.company.com/application

What does this do?

Forklift

Declare your dependencies: 


								# myapp/forklift.yaml
								services:
									- postgres
									- elasticsearch

Run the container: 


								forklift containers.company.com/application serve

Configuration

  • Defaults to services on the local machine
    • PostgreSQL, Elasticsearch, Memcached, etc.
  • All configuration can be overridden: 

							# ~/.config/forklift/_default.yaml
							postgres:
								port: 5433

							# ~/.config/forklift/myapp.yaml
							environment:
								DB_LEGACY_URL: postgres://myapp:dieplz@legacy-srv/oldapp_backup_backup

Troubleshooting

How do I ssh?


						forklift containers.company.com/application sshd \
							--mount-root /tmp/myapp_root
  • sshd is installed and configured
  • sudo is enabled
  • Container filesystem is bind-mounted to the host
  • More magic to make it all work
